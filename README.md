## Как развернуть 

1) https://developer.tech.yandex.ru/services

2) Подключить API "JavaScript API и HTTP Геокодер"

3) переходим в сам апи и копируем ключик 

(*"8f2425e0-ae03-4912-8d3e-00c1d670759e" - пример*)

4) наводим на ключ и выбираем настройки. В графе "Ограничение по HTTP Referer" 
добавляем "localhost"

5) переходим в папку проекта ../templates/Main.html 

Изменяем в строчке 7 и 8 
        
    7. https://api-maps.yandex.ru/3.0/?apikey=ТУТ НАШ КЛЮЧ КОТОРЫЙ СКОПИРОВАН РАНЬШЕ БЕЗ ""&lang=ru_RU
    8. https://api-maps.yandex.ru/2.1/?apikey=ТУТ НАШ КЛЮЧ КОТОРЫЙ СКОПИРОВАН РАНЬШЕ БЕЗ ""&lang=ru_RU

6) Ждём примерно 15 минут

7) При старте проекта переходим на страничку LocalHost:5000/start там будет наш сайтик... пробуем
